Git and Github 

git pull and git push.

These commands allow the user to fetch and download content from a remote repository while simultaneously updating to a local repository with the latest changes. Similarly, the user can push local changes to the remote repository, contributing to collaborative projects.

Examples:

        git pull origin main
        git push origin main
